# Real-time-Object-Detection---YOLOv11
## Smart Waste Sorter: Real-Time Object Detection using YOLOv11

This project features a real-time object detection system designed to recognize and categorize various forms of garbage to streamline waste management. By leveraging the **YOLOv11** architecture, the model accurately identifies recyclables and non-recyclables in a live environment.

---

### Project Overview

The **Smart Waste Sorter** aims to automate the classification of waste materials to promote better recycling habits and environmental sustainability.

**Key Features:**

* **Real-Time Detection:** Processes video feeds to identify objects instantly.


* **Multi-Class Classification:** Categorizes waste into specific groups, including:
* Plastic bottles 


* Paper 


* Cans 


* Organic material 




* **High Precision:** Utilizes the latest YOLOv11 model for optimized detection speed and accuracy.

### Installation & Setup

1. **Clone the Repository:**
```bash
git clone https://github.com/kurt-matthew-villena/smart-waste-sorter.git

```


2. **Install Dependencies:**
Ensure you have Python installed, then run:


```bash
pip install -r requirements.txt

```


3. **Run the Application:**
```bash
python main.py

```
